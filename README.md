# Scambait Simulation Lab

This project is a simulated banking interface created for cybersecurity research and educational purposes.

## Purpose

The goal of this project is to demonstrate how phishing pages and social engineering interfaces are designed, and to help improve detection and awareness techniques.

## Disclaimer

This is a fictional application. It is not affiliated with any real financial institution and does not collect, store, or transmit real user data.

## Use Case

* Phishing awareness training
* Social engineering demonstrations
* Detection engineering practice (SIEM, logs, alerts)
